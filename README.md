# STICKMAN.EXE — v0.1 beta

Небольшая хоррор-игра на Phaser + TypeScript.

## Сюжет
Прекрасный день → прогулка направо → всё темнеет, появляется кровь → в конце стоит чёрный стикмен → диалог → герой сам подходит → глаза краснеют → «Привет.» → Продолжение следует.

## Управление
- Стрелки — движение
- Пробел / Вверх — прыжок
- Пробел / Enter — пролистать диалог
- F — полный экран

## Запуск
```bash
npm install
npm run dev
```

## Сборка для сайта
```bash
npm run build
```
Готовая версия — в папке `dist/`.

## Ассеты
Все спрайты и звуки сгенерированы (стикмены, птицы, деревья, кровь, музыка, хоррор-дрон, сердцебиение, джампскер). Можно заменить на свои в папке `assets/`.
